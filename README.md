# httpMissile
This program is to lunch large amount of HTTP/HTTPS requests to a specific endpoint.

## Usages
#### $ git clone https://github.com/m00nisSmiling/httpMissile.git
#### $ cd httpMissile
#### $ python requirement.py
#### $ chmod +x lunch.py
#### $ ./lunch.py -u https://www.example.com
